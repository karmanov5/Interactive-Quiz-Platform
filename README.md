# 🎓 Interactive Quiz Platform (Real-time)

Современная интерактивная платформа для проведения образовательных квизов в реальном времени. Вдохновлена Kahoot и Quizizz, но адаптирована для учебных задач с поддержкой математических формул и глубокой аналитики.

---

## ✨ Основные возможности

### 👨‍🏫 Для Учителя (Админ-панель)
*   **Конструктор квизов:** Создание вопросов трех типов:
    *   **SINGLE:** Классический выбор одного правильного ответа.
    *   **MULTI:** Выбор нескольких правильных вариантов.
    *   **TEXT:** Свободный ввод текстового ответа (регистронезависимая проверка).
*   **Редактор мультимедиа:** Поддержка изображений для вопросов и вариантов ответов (загрузка файлов или вставка из буфера обмена).
*   **Математика (LaTeX):** Нативная поддержка сложных формул через KaTeX (используйте `$` или `$$`).
*   **Лобби и Контроль:** Генерация PIN-кода сессии и QR-кода для быстрого входа. Возможность удалять (кикать) игроков.
*   **Живой мониторинг:** Отслеживание прогресса ответов учеников в реальном времени.
*   **История и Аналитика:** Полная статистика прошлых сессий с детализацией по каждому игроку и вопросу.

### 👨‍🎓 Для Ученика
*   **Быстрый вход:** Подключение по 6-значному PIN-коду без регистрации.
*   **Персонализация:** Выбор ника и анимированного аватара (Emoji).
*   **Игровой опыт:** Таймер на каждый вопрос, баллы за скорость и правильность, живые лидерборды.
*   **Glassmorphism UI:** Современный, "стеклянный" интерфейс с частицами и плавными анимациями.

---

## 🛠 Технологический стек

*   **Backend:** Node.js, Express.
*   **Real-time:** Socket.io (синхронизация состояния между учителем и учениками).
*   **Frontend:** HTML5, CSS3 (Glassmorphism), Vanilla JS.
*   **Math Rendering:** KaTeX.
*   **QR Codes:** qrcode.js.
*   **File Handling:** Multer.

## 🚀 Быстрый старт

### 1. Установка
```bash
git clone https://github.com/karmanov5/Interactive-Quiz-Platform.git
cd Interactive-Quiz-Platform
npm install
```

### 2. Запуск
```bash
# Обычный запуск
npm start

# Режим разработки (с nodemon)
npm run dev
```
Сервер будет доступен по адресу: `http://localhost:3000`

### 3. Доступ
*   **Ученик:** `http://localhost:3000/`
*   **Учитель (Пароль по умолчанию: admin):** `http://localhost:3000/admin`

## 📂 Структура проекта

*   `/public` — Статические файлы фронтенда (HTML, CSS, JS).
*   `/quizzes` — Хранилище созданных квизов и связанных изображений.
*   `/stats` — История проведенных сессий.
*   `server.js` — Логика сервера и Socket-событий.
*   `question_generation_guide.md` — Инструкция для ИИ по генерации новых вопросов.
