# Инструкция для ИИ по генерации вопросов для Квиза

Для создания вопросов к нашему квизу используйте следующий JSON-шаблон. Квиз поддерживает три типа вопросов: **SINGLE** (один ответ), **MULTI** (несколько ответов), **TEXT** (ввод текста), а также **изображения** для вопросов и вариантов ответов.

## Формат JSON

```json
[
  {
    "text": "$\sqrt{x^2} = |x|$ - это верное утверждение?",
    "image": "/quizzes/{quiz_id}/math_problem.png",
    "type": "SINGLE",
    "options": ["Да", "Нет"],
    "correctAnswer": 0,
    "time": 20
  },
  {
    "text": "Выберите правильный график функции $y=x^2$",
    "type": "SINGLE",
    "options": ["График A", "График B", "График C", "График D"],
    "optionImages": {
       "0": "/quizzes/{quiz_id}/graph_a.png",
       "1": "/quizzes/{quiz_id}/graph_b.png"
    },
    "correctAnswer": 0,
    "time": 40
  },
  {
    "text": "Какие из этих городов находятся в Европе?",
    "type": "MULTI",
    "options": ["Париж", "Токио", "Берлин", "Нью-Йорк"],
    "correctAnswers": [0, 2],
    "time": 30
  },
  {
    "text": "Назовите столицу Франции",
    "type": "TEXT",
    "correctText": "Париж",
    "time": 20
  }
]
```

## Описание типов и полей

### 1. SINGLE (Один правильный ответ)
- `type`: `"SINGLE"`
- `options`: Массив из 2-4 строк.
- `correctAnswer`: Индекс правильного ответа (от 0 до 3).
- `time`: Время на раздумье в секундах (рекомендуется 15-20).

### 2. MULTI (Несколько правильных ответов)
- `type`: `"MULTI"`
- `options`: Массив строк.
- `correctAnswers`: Массив индексов всех правильных ответов (например, `[0, 2]`).
- `time`: Время на раздумье в секундах (рекомендуется 25-40).

### 3. TEXT (Ввод текстового ответа)
- `type`: `"TEXT"`
- `correctText`: Правильная фраза или слово. Проверка на стороне сервера регистронезависимая.
- `time`: Время на раздумье в секундах (рекомендуется 20-30).

---
**Важно:** 
- Всегда возвращай только валидный JSON-массив.
- Старайся делать вопросы интересными и разнообразными.
- Для типа `TEXT` выбирай однозначные ответы, чтобы избежать путаницы.
- **Математика:** Используй `$` для формул внутри строки (например, `$x^2$`) и `$$` для выделенных формул по центру. Поддерживаются корни (`\sqrt{x}`), дроби (`\frac{a}{b}`), степени (`x^n`) и другие LaTeX-команды.
- **Изображения:** Используйте поле `"image": "/quizzes/{quiz_id}/имя_файла.png"` для вопроса и `"optionImages": { "индекс": "путь" }` для вариантов ответов.
